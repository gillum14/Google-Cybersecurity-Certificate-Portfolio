# Google Cybersecurity Professional Certificate Portfolio

Hands-on projects, security assessments, incident reports, and technical notes completed while earning the **Google Cybersecurity Professional Certificate**.

This repository documents my development across security operations, risk management, network security, incident response, threat analysis, and security documentation. Rather than using the repository only to store completed coursework, I expanded key activities into structured portfolio projects with executive summaries, findings, risk assessments, and actionable security recommendations.

## Certificate Curriculum

| Course | Focus |
|---|---|
| **1. Foundations of Cybersecurity** | Core security concepts, ethics, responsibilities, and the role of a security analyst |
| **2. Play It Safe: Manage Security Risks** | Security frameworks, controls, audits, risk management, and compliance |
| **3. Connect and Protect: Networks and Network Security** | Network architecture, protocols, firewalls, VPNs, cloud security, and network attacks |
| **4. Tools of the Trade: Linux and SQL** | Linux command-line operations, permissions, filtering, relational databases, and SQL queries |
| **5. Assets, Threats, and Vulnerabilities** | Asset classification, threat actors, vulnerabilities, malware, social engineering, and threat modeling |
| **6. Sound the Alarm: Detection and Response** | Incident detection, response lifecycle, log analysis, SIEM, IDS, and packet analysis |
| **7. Automate Cybersecurity Tasks with Python** | Python fundamentals, algorithms, regular expressions, file parsing, and security automation |
| **8. Put It to Work: Prepare for Cybersecurity Jobs** | Portfolio development, stakeholder communication, escalation, interviewing, and career preparation |
